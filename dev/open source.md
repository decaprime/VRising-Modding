# Open Source Mods

A curated list of open source V Rising mods. To add a mod, update `_data/OpenSourceMods.json` (generated from the [Thunderstore API](https://thunderstore.io/c/v-rising/api/v1/package/)) and add the mod name to the lists in `OpenSourceMods.vue`.
